Detailed Description

Class responsible for creating and managing a world graph that represents airports, airlines, and flights. It is responsible for working with the FileManager to parse data and store it in the class's respective fields After parsing the data, the class is responsbile for providing various functions to query and manipulate data related to airports, airlines and flights.
